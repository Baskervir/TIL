# 순열과 조합

## 조합
+ 순서를 고려하지 않고 특정 개수만큼 선태갛는 경우의 수

### 예시
+ `{1, 2, 3}`중 2개를 선택할 때 가능한 조합
  + `{1, 2}`, `{1, 3}`, `{2, 3}`
    + `{2, 1}`은 `{1, 2}`와 같다고 보기 때문에 제거

---

## 순열
+ 순서를 고려하여 특정 개수만큼 나열하는 경우의 수

### 예시
+ `{1, 2, 3}`중 2개를 뽑는 순열
  + `1,2`/`2,1`/`1,3`/`3,1`/`2,3`/`3,2`
    + 순서가 다르면 다른 경우로 본다

---

## 차이점
| 구분 | 조합 | 순열 |
| --- | --- | --- |
| 순서 | 상관없음 | 순서 중요 |
| 수식 | nCr = n! / r!(n−r)!	| nPr = n! / (n−r)! |
